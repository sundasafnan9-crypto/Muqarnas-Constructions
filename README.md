# 🏗️ Muqarnas Constructions - Multipage Website

A premium, multi-page architectural website designed for **Muqarnas Constructions**. This project features a seamless blend of historical aesthetics and modern web functionality.

---

## 🌟 Project Overview

This website serves as a comprehensive digital presence for an architectural and construction firm. It was built with a **client-first approach**, where every design element—from the dark-themed UI to the specific layout of the appointment system—was tailored to meet the client's exact vision.

### 🎯 Custom Client Design
The project demonstrates the ability to deliver high-quality results based on strict **client specifications**. The client requested a "grand and futuristic" feel, which was achieved through custom CSS animations, glassmorphism, and a dark luxury color palette.

---

## 🚀 Key Features

* **Multi-page Architecture:** Dedicated sections for Services, About Us, Team, and Appointments.
* **Custom Dark UI:** High-contrast design for a luxury professional look.
* **Appointment System:** A tailored interface for client bookings.
* **Modular Styling:** Separate CSS files for each module to ensure clean and maintainable code.
* **Fully Responsive:** Optimized for desktop, tablets, and mobile devices.

---

## 🛠️ Tech Stack

* **HTML5:** Structured with semantic tags for better SEO.
* **CSS3:** Custom styling including Flexbox, Grid, and Glassmorphic effects.
* **Assets:** Optimized high-resolution architectural imagery.
